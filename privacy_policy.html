JASY Smart Time Assistant

Política de Privacidad — Última actualización: 5 de julio de 2026

Esta Política de Privacidad describe cómo JASY Smart Time Assistant ("la Aplicación", "nosotros") recopila, usa y protege la información cuando utilizas nuestra aplicación. Al instalar o usar la Aplicación aceptas los términos aquí descritos.

1. Información que recopilamos

La Aplicación está diseñada con un enfoque privacy-first: toda la información se almacena localmente en tu dispositivo y nunca se transmite a servidores externos.

Los datos que la Aplicación guarda localmente son:

  • Alarmas configuradas (hora, etiqueta, recurrencia, prioridad)
  • Geocercas definidas por el usuario (nombre, coordenadas, radio)
  • Recordatorios (título, descripción, fecha, prioridad)
  • Hábitos y estadísticas de uso
  • Preferencias de la aplicación (tema, idioma, escala de texto)
  • Credenciales cifradas (si usas autenticación local) — almacenadas con flutter_secure_storage

2. Permisos del dispositivo

Ubicación Opcional

Usada exclusivamente para la funcionalidad de Geocercas: detectar cuándo el usuario entra o sale de una zona geográfica configurada. Los datos de ubicación no se almacenan permanentemente ni se envían a ningún servidor.

Notificaciones Requerido

Necesarias para mostrar alertas de alarmas y recordatorios cuando llega la hora configurada.

Almacenamiento / Archivos Opcional

Usado para exportar estadísticas en formato CSV/PDF y para reproducir sonidos de alarma desde archivos temporales.

Sensores (acelerómetro, sensor de proximidad) Opcional

Pueden utilizarse en funciones especiales de alarma (p. ej. desactivar con sacudida o acercamiento). Los datos de sensores no se almacenan ni transmiten.

3. Uso de servicios de terceros

OpenStreetMap

El mapa integrado en la sección de Geocercas utiliza tiles de OpenStreetMap. Al visualizar el mapa, tu dispositivo hace solicitudes HTTP a los servidores de OSM para descargar los tiles. Consulta la política de privacidad de OSM para más detalles.

Geolocalización por IP (fallback)

Si el GPS no está disponible, la Aplicación puede realizar una consulta anónima a ipapi.co para estimar tu ubicación aproximada a partir de tu IP pública. Esta consulta es opcional y no incluye datos personales identificables.

4. Almacenamiento y seguridad

Todos los datos de la Aplicación se almacenan localmente en tu dispositivo usando Hive (base de datos cifrada) y flutter_secure_storage (almacenamiento seguro del sistema operativo). No utilizamos servidores en la nube ni bases de datos remotas propias.

La seguridad de los datos depende en parte de las medidas de seguridad de tu dispositivo (PIN, huella dactilar, cifrado del disco).

5. Datos de menores

La Aplicación no está dirigida a menores de 13 años. No recopilamos intencionalmente información de niños. Si eres padre o tutor y detectas que un menor ha usado la Aplicación, puedes solicitar la eliminación de datos contactándonos.

6. Tus derechos

  • Acceso: puedes ver todos tus datos directamente en la Aplicación.
  • Eliminación: desinstalar la Aplicación elimina todos los datos locales automáticamente.
  • Portabilidad: la función de exportación (CSV/PDF) te permite obtener una copia de tus datos.
  • Revocación de permisos: puedes revocar cualquier permiso desde los Ajustes del sistema de tu dispositivo en cualquier momento.

7. Cambios en esta política

Podemos actualizar esta política ocasionalmente. Notificaremos los cambios significativos mediante una actualización de la Aplicación. Te recomendamos revisar esta página periódicamente.
